# Instruções para a execução

1. Abra um terminal localizado na pasta "server-socket"

2. Execute "make all"

3. Execute "make run"

4. Mantendo a execução do servidor aberta,
abra outro terminal, e abra a pasta "client-socket"

5. Execute "make all"

6. Execute "make run"
